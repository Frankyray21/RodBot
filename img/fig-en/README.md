# English manual — lesson figure crops (`img/fig-en/`)

Place **English** manual figure images here (used inside lessons), one JPG per
source page, same naming as the French `img/fig/` folder:

```
img/fig-en/p06.jpg
img/fig-en/p13.jpg
...
```

- Shown inside lessons when the site language is **EN**.
- Missing English figures fall back automatically to the French `img/fig/` crop.

**Easiest option:** add the English PDF at the repo root and ask to have these
generated automatically.
