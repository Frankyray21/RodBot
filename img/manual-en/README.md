# English manual — page images (`img/manual-en/`)

Place the **English** operator's-manual page images here, one JPG per page:

```
img/manual-en/p01.jpg
img/manual-en/p02.jpg
...
img/manual-en/p87.jpg
```

- Same naming as the French plates in `img/manual/` (`p01`…`p87`, zero-padded).
- These are shown in the in-app manual viewer when the site language is **EN**.
- If an English page is missing, the app automatically falls back to the French
  plate of the same page number, so nothing breaks.

**Easiest option:** just add the English PDF (see repo root) and ask to have these
generated automatically from it — no need to export images by hand.
